# Data Extraction Project with n8n

This project demonstrates how to use n8n to automate the process of extracting data from an API and saving it into a database or file.

## Steps:

### 1. Download and Install n8n
Visit the [n8n official website](https://n8n.io/) and follow the installation instructions.

### 2. Import Workflow
Import the workflow from `workflow/workflow.json` into your n8n instance.

### 3. Configure Nodes
Configure the nodes for API requests and storage according to your needs.

### 4. Execute the Workflow
Execute the workflow to extract and save the data.

### Example Workflow
Check the `workflow.json` file for an example of how to set up the API request and data extraction process.
